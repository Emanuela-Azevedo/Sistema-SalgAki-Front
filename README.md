# 📖 Sobre o Projeto

O **Sistema SalgAki Front-end** corresponde à camada de apresentação da aplicação **Sistema SalgAki**, sendo responsável pela interação entre o usuário e o sistema.

A interface foi desenvolvida priorizando simplicidade, organização e usabilidade, permitindo que o gerenciamento da salgaderia seja realizado de maneira intuitiva.

Este projeto integra-se ao repositório do Back-end para fornecer todas as funcionalidades da aplicação.

---

# 🎯 Objetivos

- Desenvolver uma interface moderna e intuitiva;
- Aplicar conceitos de desenvolvimento Front-end;
- Integrar a interface com a API do Sistema SalgAki;
- Melhorar a experiência do usuário;
- Praticar responsividade e organização de componentes.

---

# 🚀 Como Executar

## 1. Clone o repositório

```bash
git clone https://github.com/Emanuela-Azevedo/Sistema-SalgAki-Front.git
```

## 2. Acesse a pasta

```bash
cd Sistema-SalgAki-Front
```

## 3. Execute o projeto

Abra o arquivo `index.html` em seu navegador ou utilize uma extensão como **Live Server** no Visual Studio Code.
